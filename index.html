我的英语学习方法

发表于 2015-04-22   |   分类于 语言   |  

挣扎学习英语十年，每每初遇挫折便坚持不下去。其中固然有懒惰作祟，学习英语一直不得其法在我看来才是主因。幸而遇见“恶魔的奶爸”这号人物，通过看他与伍君仪合著的书《把你的英语用起来!》、豆瓣小站：奶爸的英语教室、知乎回答：恶魔的奶爸，总结了如下方法。

Dr.Krashen的二语习得理论

  1. input 假说。输入量太少，并非练习量太少。
  2. i+1 理论。也叫做可理解性输入假说。i 即你现在的水平，1代表比你稍高一点点难度。时刻表征输入材料满足i+1原则。
  3. narrow input 假说。输入材料的类型和难度在一定时间内要相对狭窄。
  4. 情绪机制。学习环境应该是轻松自如的、有趣的，才能实现效果的最大化。

1和2为理论的核心。

中国建筑的特征

梁思成 中国建筑的四个基本点

  • 以木料为主要构材
  • 构架制之结构原则
  • 以斗栱为结构关键

  • 不求原物长存的观念
  • 建筑活动受道德观念约束
  • 建筑布局重视等级与规制
  • 建筑艺术与技术依靠师徒传承，不依赖于书面传承
